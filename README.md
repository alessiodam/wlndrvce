wlndrvce
========

A WLAN driver for multiple USB Wi-Fi adapters working with multiple chipsets.

Quick overview
--------------
- Purpose: Probe and attach supported Atheros USB Wi‑Fi devices (AR9271) and provide firmware packaging and helper tools.
- Repo layout (high level):
  - `src/` — driver and USB handling source files
  - `wlndrvce/` — driver implementation and chipset code
  - `firmwares/` — firmware blobs and manifest
  - `tools/` — helper scripts and utilities
